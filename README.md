O presente trabalho surge no âmbito da unidade curricular de Fundamentos de Programação e tem como objetivo o desenvolvimento de uma solução algorítmica para um problema prático. De entre as propostas apresentadas, foi selecionada a Opção 8 - Jogo.
O sistema desenvolvido consiste numa recriação do clássico jogo "Snake" (Cobra). O objetivo do jogo é controlar uma "cobra" que cresce à medida que consome "comida", evitando colidir com as paredes ou com o próprio corpo.
Para além da mecânica de jogo, o sistema foi desenhado para cumprir requisitos técnicos específicos de avaliação
